# Tunarche_java


## Description du Projet

Tun'Arche Desktop est une application JavaFX .  
Elle permet aux utilisateurs et artistes d’interagir avec la plateforme Tun'Arche via une interface graphique moderne : participation à des concours, formations, événements, gestion d’œuvres, etc.

Projet réalisé dans le cadre de PIDEV 3A à **Esprit School of Engineering**.

## Installation

1. **Cloner le repository**

```bash
git clone git@github.com:Ameni-Belkhir123/Tunarche_java.git
cd tunarche-java
```

2. **Compiler le projet avec Maven**

```bash
mvn clean install
```

3. **Lancer l'application**

```bash
java -jar target/tunarche-java.jar
```

## Utilisation

- Connexion avec identifiants
- Consultation des galeries d’œuvres
- Participation à des concours et événements
- Suivi et inscription à des formations
- Suivi des blog
- Suivi et participation a des événements 
- Données en JSON traitées avec **Gson** ou **Jackson**

## Contribution

1. Fork du projet

3. Ajouter vos changements
4. Soumettre une pull request avec explication

Merci de respecter les bonnes pratiques Java : nommage clair, découpage MVC, gestion d’exceptions, etc.
